## Authentication Providers

- Facebook

## Environment Variables

- `FACEBOOK_APP_TOKEN`: Facebook App Token (`"appid|appsecret"`)
- `MONGO_URI`: MongoDB connection URI
- `JWT_SECRET`: Secure secret string for authentication.
